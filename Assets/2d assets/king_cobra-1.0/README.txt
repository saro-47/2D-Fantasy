Name:
  King Cobra
Version:

Description:
  Cobra/King Cobra sprite originally created for Stendhal.
Sprite details:
  - Orientation: orthogonal
    - N/E/S/W format for Stendhal.
    - May eventually add S/W/E/N format for RPG Maker & RPG Boss.
  - Dimensions: 96x96 & 64x64
  - Animation:
    - idle (center frames)
    - walking: 12 frames (3 per direction)
    - Animated preview @ 150ms delay.
  - PNG images optimized with indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
